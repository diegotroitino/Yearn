1. pessoa entra no site
2. cadastra um novo evento (email, nome, data/hora)
3. sistema cria um User com o email, timezone a partir da máquina local e password aleatório
4. sistema cria um Event para o User recém criado com o nome, a data/hora e a timezone igual a do User
5. sistema manda um email para o User recém criado com o password aleatório e informações sobre o Event recém criado.
6. sistema faz login automaticamente para o User e redireciona para Events List.

5. sistema pergunta e oferece as opções: sair, sair e bloquear, não sair
